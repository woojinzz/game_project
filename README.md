# AI Simulation Game

Godot Engine으로 구현한 AI 시뮬레이션 게임입니다. 20마리의 AI 에이전트가 유틸리티 AI를 사용하여 자율적으로 행동하며, 자원 경쟁과 사회적 상호작용을 관찰할 수 있습니다.

## 🎮 게임 특징

### AI 에이전트 시스템
- **20마리 에이전트**: 각각 독립적으로 의사결정
- **3가지 주요 수치**: 배고픔, 에너지, 신뢰도
- **유틸리티 AI**: 상황에 따른 최적의 행동 선택

### 행동 시스템
- **배회(Wander)**: 기본 탐색 행동
- **먹기(Seek Food)**: 자원 탐색 및 섭취
- **휴식(Rest)**: 에너지 회복
- **교역(Trade)**: 다른 에이전트와 협력
- **도망(Flee)**: 위험 상황 회피

### 시뮬레이션 요소
- **자원 시스템**: 제한된 식량 자원으로 경쟁 유발
- **충돌 시스템**: 자원 부족시 자연스러운 갈등 발생
- **사회적 상호작용**: 신뢰도 기반 협력 및 교역

## 📊 실시간 통계

왼쪽 패널에서 다음 정보를 실시간으로 확인:
- 에이전트 수
- 평균 배고픔/에너지/신뢰도
- 사용 가능한 자원 수
- 감지된 충돌 수
- 시도된 교역 수

## 🗂️ 파일 구조

```
game_project/

├── scenes/
│   ├── Main.tscn         # 메인 게임 씬
│   └── Agent.tscn        # 에이전트 씬
├── scripts/
│   ├── GameManager.gd    # 게임 매니저 (시뮬레이션 제어)
│   ├── Agent.gd          # AI 에이전트 클래스
│   ├── UtilityAI.gd      # 유틸리티 AI 시스템

│   └── StatsPanel.gd     # 통계 UI 패널
├── assets/
│   └── textures/         # 텍스처 파일
└── resources/            # 게임 리소스
```

## 🚀 실행 방법

1. **Godot Engine 설치**
   - https://godotengine.org/download/ 에서 최신 버전 다운로드
   - Windows의 경우 ZIP 파일 압축 해제 후 실행

2. **프로젝트 열기**
   - Godot Engine 실행
   - "Import" 클릭
   - `project.godot` 파일 선택
   - 프로젝트 열기

3. **게임 실행**
   - Godot 에디터에서 F5 키 또는 재생 버튼 클릭
   - Main.tscn이 메인 씬으로 자동 설정됨

## 🎯 관찰 포인트

### 유틸리티 AI 행동 패턴
- 배고픔이 높을수록 음식 탐색 행동 증가
- 에너지가 낮을수록 휴식 행동 선호
- 신뢰도가 낮을수록 도피 행동 증가

### 사회적 역학
- 자원이 충분할 때: 평화로운 탐색과 교역
- 자원이 부족할 때: 경쟁과 충돌 증가
- 신뢰도 하락시: 협력 감소, 개별 행동 증가

### 시스템 균형
- 자원 재생성으로 지속적인 시뮬레이션
- 다양한 행동 가중치로 예측 불가능한 상호작용
- 관찰자 시점으로 자연스러운 AI 행동 관찰

## 🔧 시스템 요구사항

- **Godot Engine 4.x** 이상
- **Windows 10** 이상
- **RAM**: 최소 4GB
