# 🤖 AI 시뮬레이션: 창발 현상 관찰 게임

복잡한 상호작용에서 나타나는 창발적 행동 패턴을 관찰하는 AI 시뮬레이션 게임입니다. 20마리의 자율 에이전트가 만들어내는 예측 불가능한 사회적 역학과 집단 지능을 실시간으로 체험해보세요.

## ✨ 핵심 특징

### 🧠 고도화된 AI 에이전트 시스템
- **20마리 자율 에이전트**: 각자 독립적인 의사결정과 개성
- **3차원 감정 모델**: 배고픔 🍽️, 에너지 ⚡, 신뢰도 🤝
- **유틸리티 AI**: 복잡한 상황 판단과 최적화된 행동 선택
- **학습하는 사회적 관계**: 상호작용을 통한 관계 형성과 변화

### 🌟 창발 현상 시스템
- **자발적 집단 형성**: 에이전트들의 자연스러운 그룹화
- **사회적 위계 구조**: 리더십과 영향력의 자연스러운 등장
- **문화적 행동 전파**: 성공한 전략의 집단 내 확산
- **협력과 갈등의 연쇄반응**: 개별 행동이 만드는 집단 현상

### 🎯 관찰자 중심 설계
- **5가지 행동 유형**: 배회, 먹이찾기, 휴식, 교역, 도망치기
- **실시간 창발 현상 탐지**: 8가지 창발 패턴 자동 인식
- **다양한 관찰 모드**: 자유 관찰, 에이전트 추적, 사회 네트워크 분석
- **직관적 한국어 인터페이스**: 모든 정보를 한국어로 명확하게 표시

## 📊 실시간 관찰 도구

### 왼쪽 통계 패널
- 🧬 **에이전트 현황**: 활성 에이전트 수와 전체 상태
- 🍽️ **평균 배고픔**: 집단의 생존 압박 수준
- ⚡ **평균 에너지**: 활동 능력과 피로 상태
- 🤝 **평균 신뢰도**: 사회적 결속력과 협력 가능성
- 🌾 **자원 현황**: 경쟁의 치열함 측정
- ⚔️ **충돌 감지**: 갈등 발생 빈도와 강도
- 💱 **교역 성공**: 협력과 상호부조 수준

### 관찰자 컨트롤
- ⏱️ **속도 조절**: 0.1배~3.0배 시뮬레이션 속도
- 🔍 **에이전트 추적**: 흥미로운 개체 자동 탐지 및 추적
- 📊 **창발 분석**: 실시간 패턴 분석과 현상 탐지
- 🔄 **시뮬레이션 초기화**: 새로운 실험 시작

### 8가지 창발 현상 탐지
1. 🏘️ **자발적 집단 형성**: 에이전트들의 클러스터링
2. 🚶‍♀️ **집단 이주**: 대규모 이동 패턴
3. 🏛️ **자원 독점**: 특정 개체의 자원 독점
4. 👑 **사회적 위계**: 리더십 구조 형성
5. 🌊 **문화 전파**: 행동 패턴의 집단 확산
6. 🗺️ **영역 행동**: 공간적 경계 형성
7. 🤝 **협력 연쇄**: 긍정적 상호작용 확산
8. ⚔️ **갈등 악화**: 부정적 상호작용 확산

## 🎮 조작 방법

### 기본 관찰
- **마우스**: 자유로운 화면 이동과 확대/축소
- **스페이스바**: 일시정지/재시작 토글
- **엔터키**: 관찰 모드 순환 (자유관찰 → 추적 → 분석 등)
- **숫자 1,2,3**: 빠른 속도 조절 (0.5x, 1.0x, 2.0x)

### 고급 관찰 모드
- **자유 관찰**: 전체적인 시뮬레이션 흐름 파악
- **에이전트 추적**: 특정 개체의 행동 패턴 상세 분석
- **활동 집중지역**: 상호작용이 많은 핫스팟 탐지
- **창발 현상 추적**: 새로 나타나는 패턴 실시간 감지
- **사회 네트워크**: 에이전트 간 관계 구조 시각화

## 🚀 실행 방법

1. **Godot Engine 설치**
   - https://godotengine.org/download/ 에서 최신 버전 다운로드
   - Windows의 경우 ZIP 파일 압축 해제 후 실행

2. **프로젝트 열기**
   - Godot Engine 실행
   - "Import" 클릭
   - `project.godot` 파일 선택
   - 프로젝트 열기

3. **게임 실행**
   - Godot 에디터에서 F5 키 또는 재생 버튼 클릭
   - Main.tscn이 메인 씬으로 자동 설정됨

## 🔬 창발 현상 관찰 포인트

### 개별 에이전트 행동 분석
- **생존 욕구**: 배고픔이 높을수록 적극적인 자원 경쟁
- **에너지 관리**: 피로할 때 안전한 곳에서의 휴식 선호
- **사회적 학습**: 신뢰도에 따른 협력 vs 회피 선택
- **개성 발현**: 동일한 상황에서도 개체별 다른 반응

### 집단 역학의 창발
- **자원 풍부 시기**: 평화로운 공존과 자발적 협력 증가
- **자원 부족 시기**: 경쟁 심화와 영역 다툼 발생
- **신뢰 붕괴**: 배신 경험 후 사회 전체의 협력도 급감
- **리더 등장**: 높은 성공률을 보인 개체 주변의 자연스러운 집결

### 예측 불가능한 창발 패턴
- **연쇄 반응**: 한 에이전트의 행동이 만드는 집단 전체의 변화
- **문화 형성**: 성공한 전략의 모방을 통한 집단 문화 확산
- **시간적 패턴**: 주기적으로 나타나는 협력-경쟁 사이클
- **공간적 패턴**: 자원 분포에 따른 자연스러운 세력권 형성

### 관찰자를 위한 팁
- **인내심을 가지고 관찰**: 진정한 창발은 시간이 걸립니다
- **다양한 관점에서 관찰**: 개별과 집단, 단기와 장기를 모두 고려
- **패턴 찾기**: 반복되는 행동이나 예외적인 상황 주목
- **실험적 사고**: "만약에" 시나리오를 상상하며 관찰
